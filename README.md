# Claude 插件官方指南

> 在线阅读 · [inferloop.dev/claude-plugins](https://inferloop.dev/claude-plugins)  
> 所有书目 · [inferloop.dev](https://inferloop.dev)

系统化梳理 Claude Code 插件体系——从开发工具类、代码质量、输出风格、领域专属，到 LSP 集成与外部服务接入，覆盖每一类插件的设计动机、典型场景与官方推荐写法。

## 这本书写给谁

- 用 Claude Code 已经一段时间、想把插件用起来扩展能力的工程师
- 团队要建自己的插件 / Skill 库，想先看清楚官方推荐的边界与套路
- 对 Claude Code 内部机制好奇，想透过插件视角理解它怎么扩展
